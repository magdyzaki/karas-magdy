# كيفية الاختبار مع صديقك

## الخطوات

### 1. تشغيل السيرفر
```bash
npm run dev
```

### 2. معرفة عنوان IP جهازك (على نفس الشبكة)
- **ويندوز:** افتح CMD واكتب `ipconfig` ثم ابحث عن `IPv4 Address`
- مثال: `192.168.1.2`

### 3. فتح التطبيق

| من أين | الرابط |
|--------|--------|
| **من جهازك** | http://localhost:5000 |
| **من جهاز صديقك** (نفس الواي فاي) | http://192.168.1.2:5000 (استبدل بعنوان IP جهازك) |

### 4. تسجيل الدخول

- **أنت:** ادخل رقم مثل `1111111111` واضغط دخول
- **صديقك:** ادخل رقم مختلف مثل `2222222222` واضغط دخول

### 5. بدء المحادثة

- اضغط "محادثة جديدة"
- ابحث برقم صديقك (مثل `2222222222`)
- اضغط على اسمه لبدء المحادثة
- اكتب رسالة واضغط إرسال

### 6. من جهة صديقك

- يدخل برقمك في البحث ويبدأ محادثة معك
- أو سترى المحادثة في قائمة المحادثات عندما يرد

---

## ملاحظات

- يجب أن يكون **صديقك على نفس شبكة الواي فاي** معك
- إذا كان على شبكة مختلفة (مثلاً من بيته)، استخدم [ngrok](https://ngrok.com) لنشر السيرفر مؤقتاً:
  ```bash
  npx ngrok http 5000
  ```
  ثم شارك رابط ngrok مع صديقك
