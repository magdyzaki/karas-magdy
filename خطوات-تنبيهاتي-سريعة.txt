  خطوات سريعة — تشغيل تطبيق التنبيهات على الموبايل
═══════════════════════════════════════════════════════════

تم إعداد reminders-backend للنشر على Replit ورفعه على GitHub.
كل المطلوب منك 6 خطوات بسيطة:

───────────────────────────────────────────────────────────
1) Replit — استيراد المشروع
───────────────────────────────────────────────────────────
   • ادخل: replit.com
   • Create → Import from GitHub
   • اكتب: magdyzaki/reminders-backend
   • Import

───────────────────────────────────────────────────────────
2) Replit — تشغيل
───────────────────────────────────────────────────────────
   • اضغط Run (أعلى الصفحة)
   • انتظر رسالة: سيرفر التنبيهات يعمل...

───────────────────────────────────────────────────────────
3) Replit — نسخ الرابط
───────────────────────────────────────────────────────────
   • Share أو Webview أو Open site
   • انسخ الرابط كاملاً
   مثال: https://reminders-backend-xxx.replit.app

───────────────────────────────────────────────────────────
4) Vercel — ضبط الرابط
───────────────────────────────────────────────────────────
   • vercel.com → reminders-frontend → Settings
   • Environment Variables
   • VITE_API_URL = الرابط من Replit (بدون / في الآخر)

───────────────────────────────────────────────────────────
5) Vercel — إعادة النشر
───────────────────────────────────────────────────────────
   • Deployments → آخر نشر → Redeploy

───────────────────────────────────────────────────────────
6) الموبايل
───────────────────────────────────────────────────────────
   • افتح التطبيق من الموبايل
   • سجّل الدخول — المفروض يعمل ✓
